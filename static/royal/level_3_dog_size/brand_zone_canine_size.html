ДЛЯ ВЗРОСЛЫХ СОБАК

Чтобы собака сохранила здоровье и оставалась в прекрасной форме на протяжении всей жизни, ее рацион должен включать необходимые для удовлетворения ее пищевых потребностей компоненты. При составлении рациона необходимо не только учитывать индивидуальные параметры вашего животного, но и позаботиться о правильном балансе компонентов.

У Royal Canin есть точная формула для каждой взрослой собаки. Размер животного определяется на основании «идеального» веса породы во взрослом состоянии и подразделяется на следующие категории:

  • - X-Small: 1-4 кг
  • - Mini: 1-10 кг
  • - Medium: 11-25 кг
  • - Maxi: 26-44 кг
  • - Giant: более 45 кг

Каждый из кормов Royal Canin для взрослых собак содержит более 50 нутриентов в строго определенных пропорциях, чтобы обеспечить эффективное питание для собак любого размера, возраста и породы, а также с любым образом жизни.

Крокеты каждого продукта имеют определенную форму, размер, текстуру и плотность, в соответствии с потребностями вашей собаки
